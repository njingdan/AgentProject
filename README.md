# AgentProject

## 简介

AgentProject 是一个智能扫地机器人项目的示例，旨在展示如何使用现代技术构建机器人系统。

## 功能

- 基于传感器的环境感知
- 路径规划与导航
- 实时数据监控
- 远程控制

## 技术栈

- Python
- Flask
- TensorFlow
- OpenCV

## 快速开始

1. **克隆仓库**：
   ```bash
   git clone https://github.com/njingdan/AgentProject.git
   ```

2. **安装依赖**：
   ```bash
   cd AgentProject
   pip install -r requirements.txt
   ```

3. **运行项目**：
   ```bash
   python app.py
   ```
